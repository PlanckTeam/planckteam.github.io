# Planck Team Website

The official website for the Planck Team. This site serves as our public-facing homepage and showcase portal.

Built with performance and simplicity in mind using **Astro**. Uses **Sveltia CMS** for content management.

## 🚀 Tech Stack

- **Framework:** [Astro](https://astro.build/) (Static Site Generation / SSR)
- **CMS:** [Sveltia CMS](https://sveltiacms.app/) (Blog post management)
- **Deployment:** GitHub Pages (`planckteam.github.io`)
- **Package Manager:** `npm`
